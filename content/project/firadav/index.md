**Role:** Principal Investigator (PI).
**Funding:** Athena RC Internal PoC (10k€).

We developed a prototype database containing synchronized **mmWave Radar** (TI IWR6843) and **Camera** data. This dataset is used to train **Deep Unfolding** networks and other Scientific ML models for:
* Object detection in adverse weather.
* Sensor fusion for autonomous navigation.
* [cite_start]Training lightweight AI models for embedded deployment[cite: 123].
